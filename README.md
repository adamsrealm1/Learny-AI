# Learny AI

Learny AI is a polished Groq-powered chatbot with a Python backend, animated web interface, and account-based chat memory.

It keeps the main experience focused on conversation while the account system gives each user a small personal space for synced chats, sessions, and profile state. The backend stores accounts, secure password hashes, sessions, chats, messages, and account events in SQLite, keeping the project local-first without adding an external database service.
