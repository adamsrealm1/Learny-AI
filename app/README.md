# Learny AI

Learny AI is a Groq-powered chat app with a Python backend and a custom HTML, CSS, and JavaScript interface. It is built around fast conversational responses, account-based chat history, file-aware prompts, admin controls, and deployment on Wasmer with GitHub Pages support for the public frontend.

The project has moved away from the original local JSON knowledge-file idea. Learny is now a hosted AI assistant experience with server-side accounts, rate limits, moderation controls, profile pictures, attachment handling, reCAPTCHA protection, and persistent storage through SQLite locally or Wasmer MySQL in deployment.

The frontend is designed as a dark, animated chat workspace with a collapsible sidebar, account popup, chat search, message editing and deletion, file attachments, loading state, and responsive mobile behavior.
